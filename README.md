# Salary Manager

A simple and modern web application for managing employee salary information, developed as part of the **DIGILIANS Training Program**.

## Overview

Salary Manager is a lightweight employee salary management system that allows users to add, calculate, search, and manage employee salary data with an intuitive user interface.

## Features

- **Employee Management**
  - Add new employees with detailed salary information
  - View all employees in an organized list
  - Delete individual employees or clear all data

- **Salary Calculation**
  - Calculate net salary based on base salary, bonuses, allowances, and deductions
  - Real-time salary computation
  - Visual display of calculated net salary

- **Search Functionality**
  - Search employees by name
  - Search employees by department
  - Clear search filters to view all employees

- **Modern UI/UX**
  - Responsive design that works on all devices
  - Clean and professional interface
  - Gradient background with modern styling
  - Icons and visual indicators

## Technologies Used

- **HTML5** - Structure and layout
- **JavaScript (Vanilla)** - Application logic and DOM manipulation
- **Tailwind CSS** - Modern styling and responsive design
- **Font Awesome** - Icons and visual elements

## Usage

1. Open `index.html` in your web browser
2. Fill in the employee details:
   - Full Name
   - Department
   - Base Salary
   - Bonus (optional)
   - Allowances (optional)
   - Deductions (optional)
3. Click "Calculate Net Salary" to see the computed salary
4. Click "Add Employee" to add the employee to the list
5. Use the search functionality to filter employees by name or department
6. Delete employees individually or clear all data as needed
